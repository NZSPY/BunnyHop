# Printable BunnyHop Cards

This directory contains printable card templates for the physical version of BunnyHop.

## Printing Instructions

1. Print the card sheets on cardstock (recommended: 110lb/200gsm)
2. Use a paper cutter or scissors to cut along the guidelines
3. Optional: Use card sleeves for durability
4. Card size: Standard poker size (2.5" × 3.5" or 63.5mm × 88.9mm)

## Files

- `cards.html` - HTML template for generating printable cards
- `cards.svg` - SVG vector graphics for high-quality printing
- `card-backs.svg` - Card back design

## Generating Cards

To generate printable PDFs:
1. Open `cards.html` in a web browser
2. Use browser Print function (Ctrl+P / Cmd+P)
3. Select "Save as PDF"
4. Use settings: No margins, Background graphics ON
