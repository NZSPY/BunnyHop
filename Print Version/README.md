# BunnyHop Print Version

This folder contains the print version of the BunnyHop project. It includes resources intended for physical or printable formats.

## Files

### `Bunny Hop.pages`
This file is a Pages document (Apple's word processing software) and contains the formatted content for the BunnyHop project. It includes:
- Game rules
- Game Cards
- Other printable materials
 
 You will need to print out 1 copy of the rules (page 1)
 8 copies duplexed togther of pages 2 and 3 to form the main deck of cards
 1 copy of pages 4 and 4 duplexed to get the Dog card, cat token, and the harvest basket  

### `Bunny Hop.pdf` 
A PDF version of the pages file, for those with out an access to "Pages" 
It includes:
- Game rules
- Game Cards
- Other printable materials
 
 You will need to print out 1 copy of the rules (page 1)
 8 copies duplexed togther of pages 2 and 3 to form the main deck of cards
 1 copy of pages 4 and 4 duplexed to get the Dog card, cat token, and the harvest basket
